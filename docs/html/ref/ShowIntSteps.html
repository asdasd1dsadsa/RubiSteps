ShowIntSteps

ShowIntSteps[expr, var, opts]

Show steps of integration of expr over var.

Details and Options

  • The following options can be given:
  • ComplexityFunctionStringLength @* ToStringThis option will be passed the call of Simplify at the last step.
    GeneratedParametersAutomaticDuring the integration, temporary variables for substitution will be generated. This option will specify how these symbols would be named.
    SubstitutionInformationAutomaticHere SubstitutionInformation is an option for specifying how verbose the substitution information in steps you require.
    FormatTypeStandardFormFormatType specifies what format you want to get as output in.
